## 💡 Pro Tips for Success

### NAP Consistency (CRITICAL for SEO)
Use **IDENTICAL** business information across ALL directories:
- Same business name format
- Same address format
- Same phone number format
- Same email address

Example:
```
Business Name: NovaGenAI
Address: [Exact street], [City], [Postcode], Selangor, Malaysia
Phone: +60 [EXACT_NUMBER]
Email: [SAME_EMAIL_EVERYWHERE]
```

### Batch Processing
Group similar directories:
- **AI directories session:** Do all 8 in one sitting (2 hours)
- **Malaysian directories session:** Do 5-7 at once (2 hours)
- **Startup platforms:** Crunchbase, AngelList, F6S together (1.5 hours)

### Maximize Value
- **Add images everywhere** possible (logo minimum, screenshots bonus)
- **Fill ALL optional fields** (the more complete, the higher you rank)
- **Use different descriptions** slightly tailored to each platform (avoid duplicate content)
- **Claim ownership** where applicable (verify business email)

### Track Results
- Set up **Google Search Console** to monitor traffic from directories
- Create **Google Alert** for "NovaGenAI" to track mentions
- Monitor **referral traffic** in analytics
- **Quarterly review** — update outdated info, respond to reviews

## 🚫 Common Mistakes to Avoid

❌ **Inconsistent NAP** → Hurts local SEO  
✅ Use exact same business details everywhere

❌ **Generic descriptions** → Lower rankings  
✅ Customize per platform while keeping core message

❌ **Skipping images** → Less visibility  
✅ Always upload logo minimum, screenshots when possible

❌ **Ignoring verification emails** → Listings don't go live  
✅ Check email daily and verify promptly

❌ **One-and-done mentality** → Profiles get stale  
✅ Quarterly maintenance keeps listings fresh

❌ **Rushing Product Hunt** → Waste your one launch  
✅ Save for major release/milestone
